Traffic Flow Analysis Using Power BI
Project Overview

This project analyzes urban traffic patterns using Power BI. The dataset contains more than 48,000 hourly traffic records collected from four road junctions.

The goal is to understand traffic patterns across time and locations and identify peak traffic hours.

Tools Used

Microsoft Power BI Desktop

Power Query

DAX

Dashboard Features
